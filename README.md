# Teaching-Assistant-Agentic-AI

### What this agent does:
This is a **Teaching Assistant AI Agent** designed to help:
- Teachers: manage tasks, review code, summarize materials.
- Students: get summaries, analyze code, and track work.

### How to set it up:
1. Open Google Colab  
2. Install google-generativeai  
3. Add your Gemini API key  
4. Run all cells in order  

### How to use:
Type requests such as:
- "Add task: Grade assignments"
- "Summarize this text: ..."
- "Review this code: ..."
